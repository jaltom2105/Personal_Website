Project Overview

This website itself is a live experiment in "Constructive AI Collaboration." Utilizing Google AI Studio, I explored the boundaries of natural language as a programming interface. The goal was to build a professional, multi-page portfolio that reflects my dual background in AI and Psychology without writing every line of CSS by hand.

The project served as a case study for "Iterative Prompting Models"—how a human user can steer an LLM to maintain high-level design consistency across multiple sessions while refining complex UI interactions like custom entrance animations and responsive grid behaviors.

Learning & Capabilities

  • Semantic Design: Experimented with how AI Studio interprets abstract design philosophies like "Constructivist Architecture" and translates them into Tailwind CSS and structural HTML.
  • State Persistence: Leveraged the platform's ability to "remember" my intent across turns, enabling the development of complex logic like session-based animation gating.
  • Deployment Fluidity: Integrated the AI-built frontend with modern build tools (Vite), ensuring the professional output meets industry standards for performance and SEO.
