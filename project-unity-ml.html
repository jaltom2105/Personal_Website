Project Origin

This project was inspired by watching YouTube videos of machine learning models learning to do complex tasks—like playing soccer or "Red Light, Green Light." I wanted to figure out how to replicate those results myself to get a better understanding of how machine learning actually works.

I found out that Unity has its own ML-Agents toolkit, so I'm using it to start with their examples and eventually move on to designing and training my own custom agents from scratch.

The Tech Stack

Simulation: Unity 2022.3 LTS
Training Backend: Python 3.10 & PyTorch
Hardware: NVIDIA RTX 5070 (CUDA 12.4)
Visualization: TensorBoard
