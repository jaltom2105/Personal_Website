Lead Researcher

Period

Tools

Python, PyTorch, Scikit-learn

Project Overview

This research focused on developing a predictive framework for human interaction markers using deep learning. By monitoring low-level behavior—such as dwell times, micro-movements, and interaction frequency—the model attempts to classify the cognitive state of the user.

The primary goal was to create a system that could dynamically adapt its complexity based on the user's predicted cognitive load, ensuring that the machine stays in sync with the human's information-processing capacity.

Key Insights

  • Achieved 85% accuracy in predicting state transitions within controlled lab settings.
  • Identified "latency jitter" as a high-value signal for user frustration in automated systems.
  • Proposed a new architecture for "Cognitive Synchronization" in Human-AI teams.
