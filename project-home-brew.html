Project Overview

This project combined my love for biological systems with technical precision. I built a custom IoT controller using an ESP32 microcontroller to monitor and regulate the fermentation process for home brewing.

The system uses multiple DS18B20 temperature sensors and an Tilt Hydrometer integration to stream real-time data to a cloud database, providing a live dashboard of gravity readings and temperature stability.

Technical Challenges

  • Implementing a PID control loop for precise temperature regulation in a fluctuating basement environment.
  • Managing low-power Bluetooth (BLE) signals through stainless steel fermentation vessels.
  • Building a custom Python-based analytics engine to predict final gravity based on historical fermentation curves.
